# Ticket-to-flight
This project is a turn-based economic strategy game inspired by board games age of steam and ticket to ride where players manage competing airline companies on a global map represented as a weighted multigraph. Players must strategically build their financial portfolio, purchase aircraft, and secure profitable flight routes to transport passengers while adapting to dynamic world events and progressive taxation. Success requires careful logistical planning and resource management, as players participate in high-stakes auctions for turn priority and select unique strategic abilities each round to outmaneuver their rivals. More detailed information about the game, including the rules, can be found in the "Game rules" file. The libraries and frameworks used in this project are listed in the "technologies" file.
